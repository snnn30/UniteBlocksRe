# Unite Blocks Re
## Info
- プレイ人数：1人
- ジャンル：落ち物パズル
## OverView
同色のブロックをつなげて巨大化させていく、対戦型の落ち物パズルゲームです。

シングルプレイヤーゲームで、NPCとの対戦が楽しめます。

過去にUnityで制作した作品をベースに、ゲーム性の改善と設計の見直しを行い、Godot + C#で再構築しています。
